# My Profile & Account Management

In the **School Admin dashboard**, the bottom section of the left-hand menu provides quick access to your personal account tools:

- **My Test Link**
- **Reset Password**

- **Update Profile**

## 1. My Test Link

The **My Test Link** page displays your **unique personal registration link** that you can share directly with parents or students.

### Example URL


![My Test Link Page](/my link.png)

### What This Page Shows

- Welcome message for parents
- Branded design with:
  - School logo (e.g., AVIVA)
  - Brainwonders branding
- Automatic advisor mapping:
  - Any student registering through this link is **tagged to your advisor code**

### When to Use This

- Sharing via WhatsApp or email
- During counselling sessions
- For walk-in or referral-based registrations

## 3. My Profile

The **My Profile** page shows a read-only summary of your advisor account details.

![My Profile View](/profile.png)

### Information Displayed

| Field | Description |
|-----|-------------|
| **Name** | Advisor / counsellor name |
| **Status** | Active or Inactive |
| **Contact Number** | Masked for privacy |
| **Email ID** | Masked |
| **RM Code / Advisor Code** | Your unique tracking code |
| **Test Link Code** | Same code used in your test URL |
| **Created Date** | Account creation date |
| **Links** | Advisor list & student count summary |

This page is **view-only** and cannot be edited.

---

## 4. Update Profile

Use **Update Profile** to modify your personal contact details.

![Update Profile Form](/update.png)

### Editable Fields

| Field | Required | Notes |
|-----|---------|-------|
| Advisor Name | Yes | Full name |
| Contact Number | Yes | Mobile number |
| Email ID | Yes | Used for system notifications |
| Address | Optional | Office or residential address |

Click **Submit** to save your changes.

> **Important:**  
> Your **Advisor Code / RM Code** is permanent and **cannot be changed**, as it is used for:
> - Student mapping
> - Reporting
> - Performance tracking
