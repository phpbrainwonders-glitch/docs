## Login Credentials

You log in using the following details:

- **Employee Code**  
  Your unique advisor / employee ID (provided by Brainwonders or Superadmin)

- **Password**  
  Based on your assigned **access level (L1â€“L6)**

## Standard Level-Based Passwords

| Level | Password | Typical Role |
|------|----------|--------------|
| **L1** | `zone@123` | Zone / Regional Head |
| **L2** | `circle@123` | Circle Manager |
| **L3** | `cluster@123` | Cluster Head |
| **L4** | `ro@123` | Reporting Officer / Counsellor |
| **L5** | `bwl6@123` | School / Branch Admin |
| **L6** | Custom (set by Superadmin) | Limited or entry-level access |

### Important Notes

- Login credentials are shared securely via **email or WhatsApp** during onboarding.
- Password can be changed after first login:
  **My Profile â†’ Update Profile â†’ Reset Password**
- Maximum **5 incorrect login attempts** allowed.  
  Account is temporarily blocked for **30 minutes** after that.

## Main Navigation Menu

| Menu Item | Purpose |
|---------|---------|
| **View All Client** | Search & view all registered students |
| **Download Team** | Export team performance data |
| **Download MIS Report** | Download management information reports |
| **View All L2 / L3 / L4 / L5 / L6** | View student and performance data of sub-level advisors |
| **My Test Link** | Personal registration link to share with parents |
| **Reset Password** | Change your password |
| **My Profile / Update Profile** | View or edit personal details |
| **Logout** | Securely exit the dashboard |

---

## Key Dashboard Features

- Search students by:
  - Student name
  - Parent name
  - Contact number
  - Email ID
  - Registration or test date
  - Exam status
- Dashboard summary:
  - Total tests
  - Pending tests
  - Completed tests
- Click **View Summary** on any student to see detailed reports

## First-Time Login Checklist

1. Enter **Employee Code** exactly as provided (case-sensitive)
2. Use the correct **level password**
3. Complete CAPTCHA verification
4. After login:
   - Go to **My Profile â†’ Reset Password**
   - Set a strong personal password
5. Review:
   - **Test Settings**
   - **Registration Form Settings**
