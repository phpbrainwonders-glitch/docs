# Create a School

This guide walks you through the process of creating a new school inside the Brainwonders Admin Panel.\
Follow each step carefully to ensure accurate onboarding.

## 2. School Admin Details

Fill out all required information such as:

- School Contact Number
- Address
- City, State, and Pincode
- School Coordinator Name
- Coordinator Email
- Coordinator Mobile

## 3. Admin Access

Under this section, configure:

- Whether the school should get admin login
- Username for admin
- Password
- Permissions (view report, upload student list, export data, etc.)

This is optional but recommended when schools manage their own student data.

---

## 4. Report Customization (Optional)

Here you can:

- Enable/disable report download
- Select report types
- Add footer notes or school branding
- Configure report visibility rules

This ensures the school receives customized reports aligned with their requirements.

---

## 5. Test Details

Select the assessment(s) the school will provide to students.

Information includes:

- Test Name
- Price (if applicable)
- Attempt limits
- Availability
- Default student settings

## 6. Final Step: Submit

After verifying all school details:

- Click **Submit**
- The school is instantly created and added to the system
- Admin credentials (if enabled) are generated automatically

A confirmation message will appear on successful creation.
