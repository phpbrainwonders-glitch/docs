# Super Admin Overview

The **Super Admin Panel** is the central control system for managing all institutes, students, email configurations, coupon codes, business clients, enquiries, and overall test activity across Brainwonders.

---

## Login Screen

<img
  src="/Screenshot(447).png"
  alt="Super Admin Login"
  style={{ width:"90%" }}
  title="login"
/>

To access the Super Admin Panel:

1. Enter your **Username**
2. Enter your **Password**
3. Solve the **Captcha**
4. Click **Login**

If any credential or captcha is incorrect, access will be denied for security purposes.

## School Search & Controls

The search bar supports filters like:

- School Name
- Email
- Contact Number

### Action Buttons

- **Search** – Apply filters
- **View All** – Reset and display all schools
- **Add School** – Register a new institute

---

## School List Table

The table displays all registered schools with the following fields:

| Column           | Description                                 |
| ---------------- | ------------------------------------------- |
| **School Name**  | Registered Institute Name                   |
| **Credential**   | Institute login credentials (ID & Password) |
| **Completed**    | Completed student tests                     |
| **Ongoing**      | Students currently giving the test          |
| **Yet to Start** | Students not yet started                    |
| **Total**        | Total students created                      |

Each row provides a detailed view of the institute's test statistics.
