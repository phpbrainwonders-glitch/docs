# Coupon Approval

The Coupon Approval module allows the Super Admin to **create coupons**, **manage coupon usage**, and **approve/reject coupon codes** generated for institutes.

---

## 1. Create New Coupon

To create a new coupon, navigate to:

**Applications ‚Üí Coupon Code ‚Üí Create New**

### üñºÔ∏è Add Coupon Form

![Add Coupon Form](/coupan_form_page.PNG)

### Fields Explained

| Field                 | Description                                            |
| --------------------- | ------------------------------------------------------ |
| **Coupon Name**       | Unique identifier for the coupon code.                 |
| **Coupon For**        | Select the institute or module this coupon applies to. |
| **Discount Type**     | Choose between **Price** or **Percentage** discount.   |
| **Coupon Amount**     | The amount/percentage of discount.                     |
| **Coupon Use Count**  | Number of times the coupon can be used.                |
| **Validity Date**     | Expiry date in YYYY-MM-DD format.                      |
| **Make Bulk Coupons** | Generates multiple codes for school/franchise use.     |

### Instructions (as shown on the panel)

1. Select coupon type and enter price/percentage accordingly.
2. _Coupon For_ is compulsory‚Äîset the target institute.
3. For **bulk coupons**, do _not_ enter validity or use count.
4. Bulk coupons are for schools & franchises only.

| --------------- | ---------------------------------------------- |
| **Coupon Code** | Code visible to institutes/students.           |
| **Discount**    | % or fixed amount discount.                    |
| **Institute**   | The institute associated with the coupon.      |
| **Remaining**   | Uses left (e.g., 1/1).                         |
| **Status**      | Approved / Pending / Rejected.                 |
| **Action**      | Activate / Deactivate the coupon.              |
| **More**        | Additional options (view details, edit, etc.). |

---

## 3. Approving a Coupon

To approve a pending coupon:

1. Go to the **Status** column for the coupon.
2. If status shows **Pending**, click on the action dropdown.
3. Select **Approve**.
4. Status updates to **Approved** and becomes active for usage.

---

## 4. Deactivating / Activating Coupons

- Click **Inactive** to disable an active coupon.
- Click **Active** to enable a disabled coupon.

This helps in temporarily blocking expired or misused coupons.

---

## 5. Search & Filters

The **Search Filter** allows filtering by:

- Coupon Name
- Institute
- Status (Active / Inactive / Pending)

This helps in quickly locating specific coupons among thousands.
