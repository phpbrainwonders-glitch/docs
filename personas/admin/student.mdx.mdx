# **Students Module**

The **Students** section allows administrators to manage student data, registration fields, coupons, and related academic settings.

---

# **1. Registration Form Settings**

The Registration Form Settings allow you to enable or disable fields in the student registration form.

![student]

## **Features**

- **Enable/Disable fields** such as:
  - Name + Alternative Name
  - Mother Name
  - Father Name
  - Parents Name
  - Gender
  - Age
  - Date of Birth
  - Email ID
  - Contact Number
- Each field has toggle options: **YES / NO**

---

# **2. Coupon Code List**

The Coupon Code List displays all coupons generated by the admin or school.

![Admin Overview](/coupan_list_adminx.PNG)

## **Features**

- View **Active Coupon Count**
- Columns include:
  - Coupon Code
  - Discount
  - Remaining Count
  - Validity
  - Status
- Status types: **Approved**, **Pending**, **Expired**

---

# **3. Add Coupon**

The Add Coupon page allows admins to create a new coupon manually or in bulk.

![Admin Overview](/coupan_overview.PNG)

## **Fields**

- **Coupon Name**
- **Coupon For** (School / Student)
- **Discount Type** (Price / Percentage)
- **Coupon Amount**
- **Usage Count**
- **Validity Date**
- **Bulk Coupon Option**

## **Instructions**

2. "Coupon For" is mandatory.
3. Bulk coupons:

   - Are valid for only 1 user each.
4. Bulk coupons are only for school & franchise.
