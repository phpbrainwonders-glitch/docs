## Key Settings Explained

| Setting | Required | Description | Recommendation |
|-------|----------|-------------|----------------|
| **Question Shuffle** | Yes | Randomizes question order | Enable to prevent copying |
| **Answer Shuffle** | Yes | Randomizes answer options | Enable for fairness |
| **Show Domain** | Yes | Displays domain name during test | Usually Disabled |
| **Show Sub-Domain** | Yes | Displays sub-domain name | Usually Disabled |
| **Custom Exam Name** | Optional | Overrides default test name | Useful for sessions/year |
| **Custom Instruction Content** | Optional | Custom instructions shown before test | Add school-specific rules |
| **Redirect user after test** | Yes | Redirect to custom landing page | Enable for counselling pages |
| **Redirect URL** | Conditional | URL student is redirected to | School website / booking page |
| **Redirect after seconds** | Yes | Delay before redirect | 5â€“10 seconds |
| **Custom Thank You Content** | Optional | Message shown after test | Add contact & next steps |
| **Redirect to Test Report** | Yes | Directly open report after test | Enable for instant access |
| **Add Career Pathways in Report** | Yes | Includes career pathways in PDF | Highly recommended |
| **Registration Link (Slug)** | Yes | Public test URL slug | Lowercase + hyphens only |
| **Email Settings (CC/BCC)** | Optional | Sends test/report emails | Add counsellors/parents |
