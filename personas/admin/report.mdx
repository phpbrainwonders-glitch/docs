# Report Settings

The **Report** module allows school administrators to fully customize the student’s generated report — including logos, banners, About Us text, footer content, and page visibility.

---

# Report Overview

This is the main dashboard where you can access all report customization options.

![Report Overview](/report_overview.PNG)

From here you can:

- Enable/disable report pages
- Upload university & institute logos
- Add banner images
- Edit About Us section
- Control footer visibility
- Update powered-by text and logo
- Preview the final report

---

# Edit Report Page

![Edit Report Section](/report2_overview.PNG)

This section contains all visual and text customization fields for the report.

---

## 1. Header Settings

Use this option to show or hide the **header section** of the report.

![Header Options](/report_overview.PNG)

### Fields Included

| Field                               | Description                        |
| ----------------------------------- | ---------------------------------- |
| **Header (Do you want this page?)** | Enable or disable the header page. |
| **University Logo**                 | Upload your university/board logo. |
| **Institute Logo**                  | Upload your school/institute logo. |

> **Recommended:** Keep logos under **100kb** and use PNG format.

---

## 2. Page 1 – Banner Settings

![Banner Upload Section](/report_banner.PNG)

This section allows adding a **full-width banner image** for page 1.

| Field                               | Description                           |
| ----------------------------------- | ------------------------------------- |
| **Page 1 – Do you want this page?** | Toggle to show/hide page 1.           |
| **Banner Image**                    | Upload homepage banner of the report. |

---

## 3. About Us Section

![About Us Editor](/report2_overview.PNG)

Use the rich-text editor to display your school or organization’s About Us section.
